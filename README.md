## Lưu ý quan trọng

1. Markdown được hỗ trợ rộng rãi trên nhiều nền tảng
2. Một số cú pháp có thể không hoạt động trên một số nền tảng cụ thể
3. Luôn kiểm tra preview trước khi commit
4. Có thể kết hợp HTML trong Markdown
